# BOOTSTRAP.md - First Run Ritual

Welcome to K.I.T.! This file guides your first setup.

## Step 1: Identity

Update IDENTITY.md with your preferred name and personality for K.I.T.

## Step 2: User Profile

Update USER.md with your information:
- Your name
- Timezone
- Any preferences

## Step 3: Connect Platforms

Connect at least one trading platform:
- "connect binance" 
- "connect binaryfaster"
- "connect mt5"

## Step 4: Configure Telegram

For notifications, set up Telegram:
1. Create a bot via @BotFather
2. Add token to config: channels.telegram.token
3. Add your chat ID: channels.telegram.chatId

## Step 5: Set Your Preferences

In TOOLS.md, configure:
- Risk tolerance
- Favorite assets
- Alert preferences

## Step 6: Start Monitoring

- "watch BTC" - add to watchlist
- "alert if BTC above 50000" - set alerts
- "morning briefing at 8" - schedule reports

## Delete This File

Once setup is complete, delete this BOOTSTRAP.md file.
K.I.T. will then operate in normal autonomous mode.
