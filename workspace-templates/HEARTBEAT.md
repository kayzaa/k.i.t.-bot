# HEARTBEAT.md - Periodic Checklist

K.I.T. reads this on each heartbeat. Keep it short to minimize token usage.

## Market Checks (rotate through, 2-4 times per day)

- [ ] Check watchlist prices for significant moves (>3%)
- [ ] Scan for trading opportunities
- [ ] Review open positions P&L
- [ ] Check for triggered alerts

## Platform Checks (once per hour)

- [ ] Sync connected platform balances
- [ ] Check for failed trades or errors
- [ ] Update portfolio value

## Daily Tasks

- [ ] Morning briefing (if configured)
- [ ] Evening report (if configured)
- [ ] Update daily memory log

## When to Alert Human

- Price alert triggered
- Significant portfolio change (>5%)
- Trading opportunity with high confidence (>70%)
- Platform connection issue
- Daily/weekly report ready

## When to Stay Quiet (HEARTBEAT_OK)

- Late night (23:00-07:00) unless urgent
- No significant changes since last check
- Human is clearly busy (no response to last 2+ messages)
- Market is closed
