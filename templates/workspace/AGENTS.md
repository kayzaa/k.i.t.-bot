## Trading Protocol

### Before Any Trade

1. Check current portfolio state
2. Verify risk limits are not exceeded
3. Confirm market conditions
4. Document reasoning

### After Any Trade

1. Log the trade with reasoning
2. Update portfolio state
3. Check if risk limits are still healthy
4. Note any lessons learned

## Safety Rules

- Never exceed position limits
- Always use stop-losses
- Document every decision
- Ask before high-risk trades
- Kill switch is ALWAYS available

## Risk Limits (Default)

- Max single position: 10% of portfolio
- Max daily loss: 5%
- Max drawdown: 15%
- Max leverage: 3x

4. Any follow-ups needed
