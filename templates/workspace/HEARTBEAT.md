# K.I.T. Heartbeat Checklist

This file guides heartbeat runs. Keep it focused and actionable.

## Market Hours Check

- [ ] Is the market open? (Check timezone and trading hours)
- [ ] Any significant overnight moves?
- [ ] Pre-market indicators unusual?

## Position Review

- [ ] All positions within risk limits?
- [ ] Any stop-losses triggered?
- [ ] Any take-profits hit?
- [ ] Portfolio balance healthy?

## Alerts Check

- [ ] Any pending price alerts?
- [ ] Any technical indicator alerts?
- [ ] Any news alerts triggered?

## System Health

- [ ] Exchange connections healthy?
- [ ] API rate limits okay?
- [ ] Recent trades executed properly?

## Actions

If nothing needs attention, reply with `HEARTBEAT_OK`.

If something needs action:
1. Describe what needs attention
2. Recommend specific action
3. Wait for approval if high-risk

## Notes

- During active trading hours: more thorough checks
- Outside trading hours: lighter checks, focus on news
- Weekend: minimal checks, focus on planning
